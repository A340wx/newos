Usage Notes

    Not much to write about here. You can't do a whole lot with the system as of yet. There are a few things you should know about though:

  • F12 reboots the system. I'll get rid of it later, but it's useful right now
  • Print Screen drops you into the kernel debugger
  • Hook up a serial line at 115200 n81 no flow to com1 and you'll see the kernel debugger
